## Команды

- Установка зависимостей

```
npm ci
```

- Сборка и запуск проекта

```
npm run start
```

- Запуска проекта в режиме разработки

```
npm run dev
```

- Сборка проекта

```
npm run build
```

- Сборка и запуск проекта. Команда из шаблона Vite

```
npm run preview
```
